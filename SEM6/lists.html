Definition

HTML:

HTML Stands for Hypertext Markup Language, a standardized system for tagging text files to achieve font, colour, graphic, and hyperlink effects on World Wide Web pages.

CSS:

CSS Stands for Cascading Style Sheets is a style sheet language used for describing the presentation of a document written in a markup language.

Javascript:

Javascript is an object-oriented computer programming language commonly used to create interactive effects within web browsers.
